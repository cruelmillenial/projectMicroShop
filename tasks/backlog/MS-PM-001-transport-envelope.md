# MS-PM-001 — Define Transport Envelope

## Status
Backlog

## PM-Level Purpose

Define the first hard constraint set for Project MicroShop: the transport envelope.

This constraint governs module size, module weight, doorway compatibility, box-truck loading, palletization, crane usefulness, storage geometry, and deployment sequencing.

## Scope

Define:

- maximum module dimensions
- maximum module weight
- residential doorway and hallway assumptions
- box truck loading assumptions
- pallet compatibility assumptions
- two-operator handling limits
- crane-assisted handling requirements
- loose-parts minimization rules

## Outputs

Expected outputs:

- Constraint Set 01: Transport Envelope
- module sizing rules
- preliminary transport assumptions
- open questions requiring field measurement

## Notes

This task was queued by the Microshop PM Node as the first foundational constraint task.
