## Alias Doctrine

- Public documentation uses stable engineering aliases.
- Seller-specific SKU, order ID, tracking number, and other unique commercial identifiers remain private.
- If serial numbers or unique factory markings are later found, record them only in a private inventory system.
- Repository filenames, CAD models, and experiment notes should use the aliases above.

## Required Measurements

For lenses:

- outside diameter
- center thickness
- edge thickness
- clear aperture
- focal behavior
- apparent coating
- surface defects
- preferred orientation

For prisms:

- length, width, and height
- wedge angle if measurable
- clear aperture
- beam deviation behavior
- dispersion behavior
- surface defects
- preferred mounting faces

## Handling Notes

- Avoid bare-finger contact with optical surfaces.
- Use air and optical-safe cleaning methods only when necessary.
- Store individually in soft, non-shedding protection.
- Do not use hard point clamps.
- Mark orientation on holders, not directly on optical surfaces.
