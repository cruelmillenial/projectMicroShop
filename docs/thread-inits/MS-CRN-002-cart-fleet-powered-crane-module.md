# THREAD INIT: Cart Fleet / Powered Crane Module

## Purpose

This thread is for the longer-term concept of making the shop crane a functional module in the cart fleet.

This is separate from the immediate KISS lathe-lift use case and separate from basic mechanical upgrades.

## Project Context

The Harbor Freight / Pittsburgh 2-ton foldable shop crane has already completed the original lathe lift mission.

Long term, the crane may become a modular shop material-handling appliance integrated with a cart platform.

The cart could carry the infrastructure needed for powered actuation, rigging storage, chocks, cribbing, stabilizers, and possibly ballast.

## Core Concept

The crane module should support controlled lifting and positioning in a constrained residential microshop.

It should not become a giant industrial fantasy build.

Priorities:

1. safety
2. stability
3. controlled lifting and lowering
4. compact storage
5. serviceability
6. floor protection
7. modularity
8. cost control

## Powered Actuation Doctrine

When discussing pneumatic infrastructure in this context, assume air-over-hydraulic unless explicitly stated otherwise.

Do not interpret pneumatic as a straight pneumatic lifting cylinder.

Candidate powered systems:

- drop-in air-over-hydraulic long ram
- remote air-over-hydraulic pump and single-acting cylinder
- electric hydraulic power unit
- 12 V or 120 V HPU, depending architecture
- pendant control
- manual emergency lowering

The load should be controlled by hydraulic components, not by compressible air as the lifting medium.

## Hydraulic Behavior Desired

Ideal behavior:

- button or lever up raises slowly and predictably
- lowering is metered and gentle
- hands-off state holds load
- power loss does not cause uncontrolled lowering
- hose failure does not create a free-fall condition
- emergency manual lowering exists
- hoses are routed and protected
- pump, reservoir, battery, compressor, and ballast are mounted low

Key components to consider:

- single-acting cylinder
- pump or HPU
- reservoir
- relief valve
- check valve
- load-holding or counterbalance valve near cylinder if appropriate
- metered lowering valve
- pressure gauge
- hose management
- pendant or control box
- fuse, disconnect, enclosure, and strain relief for electrical systems

## Cart Architecture Ideas

The cart module may carry:

- hydraulic pump or HPU
- battery or AC inlet
- charger if battery powered
- compressor if air-over-hydraulic
- hose reel or controlled hose loop
- rigging box
- slings
- shackles
- chocks
- cribbing
- floor pads
- outrigger arms
- stabilizer feet
- ballast mounted low
- documentation and inspection checklist

## Design Constraint

The cart should not make the crane more annoying for basic use.

Prefer a staged path:

1. stock or near-stock crane
2. mechanical upgrades
3. better casters and stability system
4. crane-compatible cart storage or docking
5. powered ram or HPU module
6. integrated crane cart only if the prior steps prove useful

## First Task

Help define a staged architecture for a powered crane cart module, separating realistic near-term upgrades from future v2 design work.
